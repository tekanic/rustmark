PDF (Print Dialog) No dependencies

              
Opens the system print dialog. Choose "Save as PDF" to export. May produce blank pages between content on some systems.

DOCX Checking…

              
Exports a Word document via Pandoc. Uses your system Pandoc if installed, otherwise auto-downloads Pandoc on first use. Preserves headings, tables, code blocks, footnotes, and reads YAML front matter for document metadata.

PDF via Pandoc Checking…

              
High-quality PDF via Pandoc + a LaTeX engine. Uses your system LaTeX if installed, otherwise auto-downloads Tectonic (a self-contained modern LaTeX engine) on first use.
